Self Hosted

In order to set up the bot, you will first need to clone the repository. Next, you will need to have ffmpeg and python3 with the poetry package installed. Now, create a folder called config and create two files - one named bot.yaml and the other named streams.yaml. In bot.yaml you can enter your discord bot's token like so:
token: "abc123" prefix: "bot" # Optional Optionally, create the predefined streams in streams.yaml like so: lofi: "1": "https://www.youtube.com/watch?v=5qap5aO4i9A" "2": "https://www.youtube.com/watch?v=-5KAN9_CzSA" "3": "https://www.youtube.com/watch?v=l7TxwBhtTUY" "4": "https://www.youtube.com/watch?v=PbkmdRX3NL4" "5": "https://www.youtube.com/watch?v=-OXi35Q9_Rg" "6": "https://www.youtube.com/watch?v=I9m-1iiTjuY" To run the bot simply enter the following commands in your terminal: poetry update poetry shell poetry run start
